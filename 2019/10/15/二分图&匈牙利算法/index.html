他主要是有两个顶点集，然后顶点之间不能连接起来。？。。
怎么判断它是二分图？。。
先将一个一个点标记为1（或者0）然后与它连接起来的为标记的的就标记为0（或者1），如果该点已经标记，则判断他是否和顶点的值相同，相同则不为二分图，反之则是是。

bool find(int x) {
int j=1;
for(; j<=n; j++) {
if(map[x][j]&&!visit[j]) {
visit[j]=true;
if(!match[j]||find(match[j])) {
match[j]=x;
return true;
}
}
}
return false;
}

que.pop();
int i=1;

if(!visit[i]) {
que.push(i);
